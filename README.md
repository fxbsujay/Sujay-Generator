# Sujay-Generator
一个简单的j代码生成器，可以生成java代码，前端js，vue等语言代码，使用freemarker模板渲染，暂时只支持mysql数据库，后面会逐渐更新

freemarker 模板语法  ---------> [http://freemarker.foofun.cn/dgui_template_overallstructure.html]()

打开后访问http://localhost:8001/
